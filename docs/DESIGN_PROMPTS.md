## 🛠️ 4. FOYDALANISH BO'YICHA AMALIY QO'LLANMA

### A. Midjourney / DALL-E 3 bilan UI Mockup yaratish:
1. Yuqoridagi promptlarni **Discord → Midjourney bot**ga `+imagine` buyrug'i bilan yuboring.
2. Yoki **ChatGPT (Plus)** da DALL-E 3 orqali ishlatiladi.
3. Natijani yuklab, **Figma** ga import qiling va CSS dizayn tizimingiz uchun namuna sifatida ishlating.

### B. Runway Gen-3 / Luma bilan Video yaratish:
1. [lumalabs.ai/dream-machine](https://lumalabs.ai/dream-machine) yoki [runwayml.com](https://runwayml.com/) ga o'ting.
2. **Text-to-Video** rejimini tanlang va promptni kiriting.
3. `1280x720` yoki `1920x1080` sifatda yuklab, `public/media/` papkasiga qo'ying.

### C. React ilovasida Video / GIF ishlatish kodi:
```jsx
// Scenario natijasiga qarab video ko'rsatish
function OutcomeVideo({ outcomeType }) {
  const videos = {
    collision: '/media/scenario-collision.mp4',
    priority_violation: '/media/scenario-priority.mp4',
    correct: '/media/scenario-correct.mp4',
  };

  return (
    <div style={{ borderRadius: 16, overflow: 'hidden', border: '1px solid rgba(255,255,255,0.08)' }}>
      <video autoPlay loop muted playsInline style={{ width: '100%', display: 'block' }}>
        <source src={videos[outcomeType]} type="video/mp4" />
      </video>
    </div>
  );
}

// GIF badge ko'rsatish
function OutcomeBadge({ isCorrect }) {
  return (
    <img 
      src={isCorrect ? '/media/badge-correct.gif' : '/media/badge-collision.gif'}
      alt={isCorrect ? "To'g'ri javob" : "Xato"}
      style={{ width: 80, height: 80 }}
    />
  );
}
```

### D. Video optimizatsiya (katta fayllar uchun):
```bash
# MP4 faylni web uchun optimallashtirish (FFmpeg)
ffmpeg -i input.mp4 -vcodec libx264 -crf 23 -preset slow -movflags +faststart output.mp4

# GIF ni WebP animatsiyaga o'girish (kichikroq hajm)
ffmpeg -i input.gif -c:v libwebp -lossless 0 -q:v 70 output.webp
```
